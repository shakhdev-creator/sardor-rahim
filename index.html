🔥 Firebase Realtime Database

Firebase ulansa — istalgan qurilma, brauzer, Telegram dan kirganingizda bir xil data ko'rinadi.

Qanday sozlash:
1. console.firebase.google.com → yangi project
2. Realtime Database → Test mode da yaratish
